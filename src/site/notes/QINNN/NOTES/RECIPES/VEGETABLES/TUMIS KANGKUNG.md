### <span style="color:orange">Bahan Utama</span>
- <span style="color:#4A90E2">Kangkung</span> 2 ikat, petik daunnya  
- <span style="color:#4A90E2">Bawang putih</span> 4 siung, iris tipis  
- <span style="color:#4A90E2">Bawang merah</span> 3 siung, iris tipis  
- <span style="color:#4A90E2">Cabai merah besar</span> 2 buah, iris serong (opsional)  
- <span style="color:#4A90E2">Cabai rawit</span> 5 buah, iris halus (opsional, untuk pedas)  
- <span style="color:#4A90E2">Saus tiram</span> 1 sdm  
- <span style="color:#4A90E2">Kecap asin</span> ½ sdm  
- <span style="color:#4A90E2">Kecap manis</span> ½ sdm  
- <span style="color:#4A90E2">Gula pasir</span> ¼ sdt  
- <span style="color:#4A90E2">Garam dan penyedap</span> secukupnya  
- <span style="color:#4A90E2">Minyak</span> 3 sdm untuk menumis  
- <span style="color:#4A90E2">Air</span> 50 ml  
### <span style="color:purple">Tambahan Bumbu Rempah (Opsional)</span>
- <span style="color:#4A90E2">Lengkuas</span> 1 ruas kecil, geprek  
- <span style="color:#4A90E2">Daun salam</span> 1 lembar  
- <span style="color:#4A90E2">Terasi bakar</span> ½ sdt  
- <span style="color:#4A90E2">Lada putih bubuk</span> ½ sdt  
- <span style="color:#4A90E2">Kaldu jamur atau udang kering halus</span> sedikit  
### <span style="color:green">Langkah</span>
<span style="color:gray; font-style: italic">
1. Panaskan minyak di wajan, tumis bawang merah, bawang putih, dan cabai sampai harum serta agak kecokelatan.<br>
2. Masukkan lengkuas, daun salam, dan terasi bila pakai, tumis hingga aromanya keluar.<br>
3. Tambahkan saus tiram, kecap asin, kecap manis, gula, garam, dan penyedap. Aduk rata.<br>
4. Tuang air, aduk, lalu masukkan kangkung.<br>
5. Tumis cepat dengan api besar sampai kangkung layu dan bumbu meresap.<br>
6. Koreksi rasa, angkat, dan sajikan hangat.
</span>

### <span style="color:teal">Tips</span>
> <span style="color:gray; font-style: italic">Gunakan api besar agar aroma smokey keluar dan bumbu cepat meresap.</span>
>
>> <span style="color:gray; font-style: italic">Jangan tumis terlalu lama supaya kangkung tetap hijau dan renyah.</span>
>>
>>> <span style="color:gray; font-style: italic">Untuk rasa lebih gurih, tambahkan sedikit ebi halus saat menumis bumbu.</span>
